* TCP 是什么？
  * TCP 面向连接的，可靠的传输协议
  * 连接是什么意思？
    * 客户端C和服务端S想要交流（交换数据），就需要连接对方
  * 可靠又是怎么说？
    * 当任意一端想要断开连接时，需要"四次分手"，使得双方安全断开，可谓可靠
    
* TCP 三次握手
  * C 发送含有SYN的报文 -> S  
    S 发送ACK报文 -> C  
    C 发送 ACK 报文 -> S  
    此时建立连接
    
* TCP 四次分手
  * C 发送含有FIN的报文 -> S  
    S 发送 FIN + ACK 报文 -> C  
    S 发送剩下的数据 -> C  
    S 发送 FIN 报文 -> C  
    C 发送 ACK 报文 -> S  
    此时断开连接
